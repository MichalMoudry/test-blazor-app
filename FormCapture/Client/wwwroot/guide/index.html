Initial application setup

This part of the documentation describes how to initially set up FormCapture application.

Steps:

  1. Create a workflow (as per this section)
  2. Create a capture application
  3. Add a new document template
  4. Associate a created workflow from step 1 to a existing capture application

After completing these steps, you will be able to start a queue where uploaded images will be processed.

User registration

To register a user, you need to visit /register page of the app where registration form exists. First 3 input fields are for email (which is used to identify the user), password and its confirmation. Last field is for selecting which role the user will have inside the system. You can see the form in the picture n. 1.

Role descriptions:

  • Admin - Admin account has access to a lot of operations regarding items within the app. For example admin has access to user interface for creating capture apps.
  • User - User has access to limited functionality and exists only for processing images.

After you have succesfully completed registration, then you will be sent to login page.

User login

To login into the app, you need to visit /login page or you can automatically navigate to it. The login form consists of 2 fields for email and password (see picture n. 2). If you are successful, then you will be navigated to a page with a list of capture applications.

User logout

To logout of the application you have to click the logout button, that is located in the top right corner of the screen (see picture n. 3).

Workflows

Workflows are used to add additional tasks to each step in image processing with the help of JavaScript. If you click on the link Workflows in side menu, then you will be navigated to a page with the list of existing workflows (as seen below) and a button, that will navigate you to a workflow creation page.

Add a new workflow

To create a new workflow, you need to click on a Add a new workflow button on the page with the list of existing workflows, then you will be navigated to the workflow creation page as seen below.

On this page you can see 4 boxes, where you can add your own JavaScript tasks to each step in image processing, but they are not required to create a workflow. Only required field is the one for workflow name.

Edit an existing workflow

To edit an existing workflow, you need to click on an item in a list with workflows. There are two list with them, one is shown in the picture n. 4 and the other one is on a page with capture application details. After that, you will be navigated to a page where you can edit the specific workflow (see picture n. 6).

This page is simillar in functionality to the page for creating a new workflow.

Workflow tasks

If you want to add a custom task to each step in image processing, then you can do that through workflow tasks. When you click on Tasks link in side menu, you will be navigated to a page with a list of existing tasks and a button to add a new one (see picture n. 7).

Add a new workflow task

To add create a new workflow task you need to navigate to /formcapture/tasks/add page through Add a new task button. On the page there is a form with 3 fields (see picture n. 8), that are required to successfully create a new task.

Edit an existing workflow task

To edit a workflow task, you need to click on a row of the table where those tasks are displayed (see picture n. 7). After that you will be navigated to a page, which contains a form for editing selected workflow task as displayed below.

Capture apps

Main page of FormCapture application consists of a list of your capture applications, or applications that your account has been added to as seen below.

Adding a new capture app

To add a new capture application, you need to navigate to /formcapture/apps (note: also it is FormCapture's main page) where you can find a Add a new app button, which opens up a modal window with a form for adding a new capture application (see picture n. 11). If you fill that for successfully, a new app will appear in the list.

Edit an existing capture app

You can edit or delete applications on the details page (see picture n. 12), which contains 2 button (marked in a red square in picture), one will display modal windows for editing app's information, the other button is for deleting said app. To access app's details page, you need to click on a item in a table displayed on /formcapture/apps page (as seen on picture n. 10).
